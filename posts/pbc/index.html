Traits

Specially designed for TPMS

  1. Automatically adapt to the constraint type of dian/bian/mian,
  2. no need to manually select set REALIZE THE AUTOMATION OF MIAN, DIAN, AND BIAN SETTINGS
  3. Solid/shell type support is supported in cases
  4. faces, edges, and points can be incomplete

Note

It is necessary to flip the cube to ensure the existence of the face edge point at the minimum value of the coordinate system
